Installationshinweis

Sollten sie Windows XP nutzen, so beachten Sie, dass der Windows-interne Unzipper beim Auspacken von verinice u.U. nicht alle Dateien verarbeitet. Wenn Sie beim Auspacken eine Meldung über angebliche verschlüsselte Dateien erhalten, verwenden Sie daher bitte ein anderes Zip-Programm wie z.B. 7-Zip, WinRAR, WinZIP oder dergleichen.
